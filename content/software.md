        ## Werkzeuge und Werkzeugkasten

        ### Personal website tools

        - This site is built with [Hugo Blox](https://hugoblox.com/) (and was
          previously built with [jemdoc](http://jemdoc.jaboc.net), a tool for
          creating clean, simple-to-maintain websites).
        - See the [create_pubs.py](/projects/create-pubs/) script — a tool to
          scrape publications from Google Scholar via a Tor service.

        ### Statistics and Machine Learning

        - [IC toolbox](/projects/ic-toolbox/) — model-order selection for
          high-dimensional, low-sample-support correlated data.
        - [SVM toolbox](https://github.com/cjlin1/libsvm/tree/master/) —
          support vector machine implementation in C, Python, and Matlab, by
          Chih-Jen Lin.
        - [EEG NN and Gait NN](/projects/eeg-gait-nn/) — example uses of
          convolutional neural networks on EEG and gait datasets.

        ### Optimization

        - [OR-Tools](https://developers.google.com/optimization/) —
          open-source package for solving various operations-research-related
          optimization problems.
        - [TSP solver](/projects/tsp-solver/) — a simple module and some
          Jupyter notebooks for playing with simulated Traveling Salesman
          problems.
        - [MISTWF toolbox](/projects/mistwf-toolbox/) — Matlab minimum
          increment space-time water-filling algorithm and test bed.

        ### Filtering and Estimation

        - [EKF/UKF Kalman filtering toolbox](http://www.lce.hut.fi/research/mm/ekfukf/)
          — Matlab filtering toolbox, visualization, and various dynamic
          models, by Simo Särkkä and Jouni Hartikainen.
